📋 邮件源文本

从邮件客户端查看原始邮件源代码，复制完整内容粘贴到下方进行分析（自动处理中文乱码）

如何获取邮件源代码：

  • Gmail：打开邮件 → 点击右上角 ⋮ → 选择「显示原始邮件」
  • Outlook (网页)：点击 ⋯ → 「查看」→「查看邮件源」
  • Outlook (客户端)：双击打开 → 文件 → 属性 → 邮件头信息
  • Apple Mail：选中邮件 → ⌘+Option+U 显示原始邮件
  • Thunderbird：选中邮件 → Ctrl+U 查看源代码
  • Foxmail / QQ邮箱：右键邮件 → 「查看原始信息」

关于中文乱码：

  • 主题和发件人中的 =?UTF-8?B?...?=、=?GBK?Q?...?= 会被自动解码
  • 支持 Base64 (B) 和 Quoted-printable (Q) 两种编码方式
  • 支持 UTF-8、GBK、GB18030、Big5、ISO-8859-1 等常见字符集

关于报告：

  • 点击「📄 生成报告」下载独立的 HTML 报告文件
  • 报告中的各模块（除概览外）默认折叠，可点击展开查看详情
  • 报告可保存为 PDF（浏览器 → 打印 → 另存为 PDF）

关于复制：

  • 「安全认证」板块中每个邮箱/域名/IP 旁边都有 📋 复制按钮
  • 点击即可复制到剪贴板（自动去除引号），按钮会变成 ✅ 反馈 1.2 秒
